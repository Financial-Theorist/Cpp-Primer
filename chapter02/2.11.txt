Exercise 2.11: Explain whether each of the following is a declaration or a
definition:

	(a) extern int ix = 1024;
	(b) int iy;
	(c) extern int iz;

by Faisal Saadatmand

(a)	definition. Note: it is an error to override extern with an initializer if
	the definition was inside a function (p. 45) 
(b) definition, uninitialized
(c) declaration
